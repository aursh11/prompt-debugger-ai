# Prompt Debugger & Improver AI

An AI-powered web application that analyzes low-quality prompts and rewrites them into high-quality, professional prompts with clear explanations.

## Features
- Identifies issues in unclear prompts
- Improves prompt structure and clarity
- Explains why the improved prompt works better
- Simple Streamlit UI

## Tech Stack
- Python
- Streamlit
- Prompt Engineering
- Large Language Models

## Use Cases
- Improving chatbot accuracy
- Training teams on prompt writing
- AI workflow optimization
